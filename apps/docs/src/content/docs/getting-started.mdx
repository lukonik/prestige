# Getting Started

:::caution
Prestige is still in active development v0. Although API is stable and core functionality is implemented, some breaking changes might stil appear.
and they might be released in patch/minor releases. We are working hard to release stable version soon (v1). Stay tuned
:::

To get started, first create Tanstack Start project, open terminal and run:

```bash
npm create @tanstack/start@latest
```

_For a hosting provided we recommend using nitro server, it has tons of hosting provided integration out of the box, like github-pages and are easy to use. Nevertherless Prestige is non-opinionated on that, as long as it works with Tanstack-Start it works with Prestige_

after installation, run Prestige create script that will automatically add Prestige to your project.

```bash
npx @lonik/create-prestige@latest
```

it will configure the project and install neccessary dependencies. After that the project is ready to go.

## Usage

run the dev `npnm run dev` and you should see the Prestige landing page with the docs navigation in the header.
navigate to the docs and you should see the introduction page. open the `introduction.mdx` file in `src/content/docs`
and update some text, you should see changes instantly in browser. in later guides we will explore the library in depth.

## Manual installation

Although we recommend usign `@lonik/create-prestige` to configure project, you can also add Prestige manually

<Steps>
  <StepItem label="Install packages" index="1">
    ```bash npm install @lonik/prestige@latest --save ```
  </StepItem>
  <StepItem label="Create content folder" index="2">
    Create `src/content/docs` folder in your project. This folder will be used
    for docs markdown pages
  </StepItem>
  <StepItem label="add PrestigeShell in _root" index="3">
    To display prestige layout like header,footer etc. add `PrestigeShell` component in `__root.tsx`
    
    ```tsx
        import { PrestigeShell } from '@lonik/prestige/ui';
        ...
         <PrestigeShell >
          <Outlet />
        </PrestigeShell>
        ...
    ```

  </StepItem>
  <StepItem label="add prestige plugin in vite config" index="4">
    In `vite.config.ts` add `prestige` plugin
    ```ts
        {
          plugins:[
            ...
                prestige({
                    title:"Prestige",
                    collections:[{
                        id:'docs',
                        items:[] // for now it is empty we will add content later
                    }]
                })
            ...
          ]
        }
    ```
  </StepItem>
</Steps>

Now prestige is fully configured to be used. For testing that everything works. create `introduction.mdx`
inside `src/content/docs`, with following content:

```md
---
title: Introduction
---

# Introduction

This is introduction page
```

and add link to the collections array in `vite.config.ts`.

```ts
    {
        ...
            prestige({
                title:"Prestige",
                collections:[{
                    id:'docs',
                    items:[{
                        label:"Introduction",
                        slug:"docs/introduction"
                    }]
                }]
            })
        ...
    }
```

run `npm run dev` navigate to `/docs/introduction` and you should see rendered introduction page.
