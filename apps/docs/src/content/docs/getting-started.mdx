# Getting Started

To get started, create prestige script:

```bash
npx @lonik/create-prestige@latest
```

follow the prompts and it will create Tanstack Start project with Prestige configuration

## Usage

Run the dev server with `npm run dev`, and you should see the Prestige landing page with the docs navigation in the header. Navigate to the docs, and you should see the show page. Open the `showcase.mdx` file in `src/content/docs` and update some text; you should see changes instantly in the browser. In later guides, we will explore the library in depth.

:::info
Adding new page, might cause infinite loop or error in the rendering process. If that happens rerun the dev server (npm run dev) again and it should fix the issue.
It will be fixed in the future.
:::
