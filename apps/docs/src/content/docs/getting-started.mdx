# Getting Started

To get started, create prestige script:

```bash
npx @lonik/create-prestige@latest
```

follow the prompts and it will create Tanstack Start project with Prestige configuration

## Usage

Run the dev server with `npm run dev`, and you should see the Prestige landing page with the docs navigation in the header. Navigate to the docs, and you should see the introduction page. Open the `introduction.mdx` file in `src/content/docs` and update some text; you should see changes instantly in the browser. In later guides, we will explore the library in depth.

## Manual installation

Although we recommend using `@lonik/create-prestige` to configure the project, you can also add Prestige manually.

<Steps>
  <StepItem label="Install packages" index="1">
    ```bash 
    npm install @lonik/prestige@latest --save 
    ```
  </StepItem>
  <StepItem label="Create content folder" index="2">
    Create a `src/content/docs` folder in your project. This folder will be used
    for docs markdown pages.
  </StepItem>
  <StepItem label="Add PrestigeShell in __root.txt" index="3">
    To display the Prestige layout, like the header, footer, etc., add the `PrestigeShell` component in `__root.tsx`.
    
    ```tsx
        import { PrestigeShell } from '@lonik/prestige/ui';
        ...
         <PrestigeShell >
          <Outlet />
        </PrestigeShell>
        ...
    ```

  </StepItem>
  <StepItem label="Add the prestige plugin in the Vite config" index="4">
    In `vite.config.ts`, add the `prestige` plugin.
    ```ts
        import {prestige} from "@lonik/prestige/vite";
        ...
        {
          plugins:[
            ...
                prestige({
                    title:"Prestige",
                    collections:[{
                        id:'docs',
                        items:[] // for now it is empty; we will add content later
                    }]
                })
            ...
          ]
        }
    ```
  </StepItem>
</Steps>

Now Prestige is fully configured to be used. To test that everything works, create `introduction.mdx`
inside `src/content/docs` with the following content:

```md
---
title: Introduction
---

# Introduction

This is the introduction page.
```

Then, add a link to the collections array in `vite.config.ts`:

```ts
    {
        ...
            prestige({
                title:"Prestige",
                collections:[{
                    id:'docs',
                    items:[{
                        label:"Introduction",
                        slug:"docs/introduction"
                    }]
                }]
            })
        ...
    }
```

Run `npm run dev` and navigate to `/docs/introduction`, you should see the rendered introduction page.
